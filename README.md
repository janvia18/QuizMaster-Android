# QuizMaster (Android)

QuizMaster is a simple Android quiz application built using **Kotlin** and **Android Studio**.  
The app presents multiple-choice questions, lets the user select an answer, and shows the score at the end of the quiz.

## Features

- Multiple-choice quiz questions
- Score calculation at the end of the quiz
- Basic, clean Android UI

## Project Structure

At a high level:

- `app/` – Main Android app module  
  - `src/main/java/...` – Kotlin source code (activities, view models, etc.)  
  - `src/main/res/layout/` – XML layout files (screens for the app)  
  - `src/main/res/values/` – Colors, strings, styles  
- `gradle/` – Gradle wrapper files  
- `build.gradle.kts`, `settings.gradle.kts` – Gradle configuration  
- `screenshots/` – Emulator screenshots for the assignment (see below)
